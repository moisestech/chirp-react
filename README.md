# Chirp

<img src="https://raw.githubusercontent.com/moisestech/chirp-react/master/public/assets/chirp_logo.png" width="150px"/>

Chirp - public social messaging application built in ReactJS.

## ⚙ Features

- React 17
- React Router v6
- React Hooks
- ES8
- PropTypes
- Light/Dark Theme
